🔎 Why Arabic Plurals Feel Hard

Sound plurals (easy, rule-based):

Masculine: add ون / ين (معلم → معلمون).

Feminine: add ات (بنت → بنات).
👉 These are predictable and not too bad.

Broken plurals (irregular):

Inside the word changes completely.

كتاب → كتب (book → books)

رجل → رجال (man → men)
👉 These are what’s breaking your heart ❤️‍🔥.

Adjectives also change to agree in gender and number.

كتاب جديد → كتب جديدة

بنت جميلة → بنات جميلات

⚡ The Good News

You don’t need to memorize all plurals at once.

Focus on the most common words — just like you don’t memorize every irregular English plural (child → children, mouse → mice) until later.

Over time, your brain starts to see patterns. Example:

Words with 3 letters like كتب / كتب often shrink in plural.


👨‍👩‍👧 Family & People

أب → آباء (ʾab → ābāʾ) = father → fathers

أم → أمهات (umm → ummahāt) = mother → mothers

أخ → إخوة / إخوان (akh → ikhwa/ikhwān) = brother → brothers / companions

أخت → أخوات (ukht → akhawāt) = sister → sisters

ابن → أبناء (ibn → abnāʾ) = son → sons

بنت → بنات (bint → banāt) = girl/daughter → girls/daughters

ولد → أولاد (walad → awlād) = boy → boys / children

طفل → أطفال (ṭifl → aṭfāl) = child → children

رجل → رجال (rajul → rijāl) = man → men

🌍 Nature & Things

بيت → بيوت (bayt → buyūt) = house → houses

شمس → شموس (shams → shumūs) = sun → suns

ظهر → ظهور (ẓahr → ẓuhūr) = back → backs

يد → أيد (yad → ayd) = hand → hands / also “strength”

وردة → ورود (wardah → wurūd) = rose → roses

🐪 Animals

أسد → أسود (asad → usūd) = lion → lions

جمل → جمال (jamal → jumāl) = camel → camels

ثعلب → ثعالب (thaʿlab → thaʿālib) = fox → foxes

سمك → أسماك (samak → asmāk) = fish → fishes

دجاج → دجاج (dajjāj) = chicken → chicken (mass noun)

🍎 Food & Drink (mostly mass nouns)

خبز → خبز (khubz) = bread → bread

ماء → مياه (māʾ → miyāh) = water → waters

لبن → ألبان (laban → albān) = milk → milks/dairy products

زيت → زيت (zayt) = oil → oil

طعام → طعام (ṭaʿām) = food → food

✨ Abstract / Other

كتاب → كتب (kitāb → kutub) = book → books

قلم → أقلام (qalam → aqlām) = pen → pens

فم → أفواه (fam → afwāh) = mouth → mouths

ضوء → أضواء (ḍawʾ → aḍwāʾ) = light → lights

حب → حب (ḥubb) = love → love (mass)

صبر → صبر (ṣabr) = patience → patience (mass)

ذهب → ذَهَب (dhahab) = gold → gold (mass)

نار → نيران (nār → nīrān) = fire → fires (but usually mass noun)

هواء → أهوية / أهواء (hawāʾ → ahwiyah/ahwāʾ) = air → airs / desires
